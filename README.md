## Objetivo:
Desenvolver uma versão **mobile first** do Instagram.

## Requisitos:
* **Node v13.8.0** 
* **Live Server**.
​  
## Iniciar aplicação:
> **`npm install -g live-server`**  
> 
> **`live-server ./[nome-da-pasta ou arquivo html]`**
